Access the world's highest quality, re-usable images of cultural heritage

Cultural heritage institutions that deliver freely licensed access to their digital collections using IIIF are listed on the IIIF website. The list includes major museums and galleries, state and national libraries, and research-led universities from around the world. In 2022, the number of cultural heritage items made available via IIIF was estimated at 1 billion.

Clicking on the link to each institution will lead you to a page that describes how to access that institution's digital collections, and how to find the manifest for a IIIF object.

The manifest is a package that contains:

  • links to all the images belonging to a digital object
  • structural metadata - how to organise and present the images
  • descriptive metadata - title, author, date, location of the physical object

The manifest is written in json, but looks like a web address. When pasted to the address bar in a browser, you can only see the code. But when pasted to a IIIF viewer or tool, the object will render.
